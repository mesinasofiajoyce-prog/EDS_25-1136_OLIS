# Predictive Maintenance System

## Project Description
This project is a machine learning–based predictive maintenance system designed to analyze industrial sensor data and predict potential machine failures. It applies data preprocessing, exploratory data analysis (EDA), and classification algorithms to detect patterns that indicate possible equipment failure.

The system supports preventive maintenance strategies by identifying risks early, reducing downtime, and improving operational efficiency.

Main workflow:
- Data loading and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection and preprocessing
- Training and evaluating machine learning models
- Visualizing failure patterns and insights
